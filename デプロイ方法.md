1. プロジェクトフォルダ内にREADME.mdを作成

2. ノートブックでスクリプトを記述して保存

```bash
uvx marimo edit notebook.py
```

3. docsフォルダ(GitHub Pagesで読み込む際に必要)にエクスポート

```bash
# アプリモード
uvx marimo export html-wasm notebook.py -o docs --mode run
# 編集可能なノートブックモード
uvx marimo export html-wasm notebook.py -o docs --mode edit
```

```bash
# ローカルサーバーを起動してテスト実行
# 起動したらブラウザでlocalhost:8000を入力
python -m http.server --directory docs
```

4. GitHubにプッシュ

5. GitHub Pageの設定でブランチのプルダウンメニューを/docsに変更
